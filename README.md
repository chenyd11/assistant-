# 抖音视频转多平台内容 - 自动化系统

> 一键将抖音视频转换为公众号、小红书、抖音图文等多平台内容

## ✨ 功能特性

- 🎬 **视频转文字**：Whisper本地转录，支持中文
- ✍️ **智能洗稿**：保留核心信息，重写表达方式
- 📱 **多平台适配**：自动生成公众号、小红书、抖音图文三版本
- 📄 **飞书推送**：自动创建飞书文档，方便协作编辑
- 🚀 **即将接入**：字流一键分发14平台

## 🚀 快速使用

### 方式1：直接发视频给我（最简单）

1. 在抖音App下载视频
2. 发送视频文件给我
3. 等待2-3分钟
4. 获得飞书文档链接（包含3个版本内容）

### 方式2：命令行处理

```bash
# 进入工作目录
cd /Users/chenyd11/.openclaw/workspace

# 一键处理
./scripts/auto_process.sh /path/to/video.mp4
```
