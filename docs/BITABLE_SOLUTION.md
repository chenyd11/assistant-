# 视频库自动化方案

## 问题
飞书API无法直接创建Bitable表格，只能创建普通文档。

## 解决方案

### 方案1：使用飞书Bitable API（推荐）
需要先在飞书中手动创建一个Bitable表格，然后通过API自动添加记录。

**步骤：**
1. 你在飞书中手动创建一个Bitable表格
2. 复制表格的URL给我
3. 我通过API自动向表格添加视频记录

**优点：**
- 真正的表格形式
- 字体大小正常
- 支持筛选、排序

---

### 方案2：使用本地数据库 + 自动生成文档
在本地维护视频库，定期自动生成飞书文档。

**已实现：**
- ✅ 本地视频库（library/）
- ✅ 编号系统（VID_2026_XXXX）
- ✅ 自动转录

**待实现：**
- 定时生成飞书表格文档

---

### 方案3：使用第三方工具
搜索到的相关GitHub项目：

1. **feishu-bot** - 飞书机器人框架
   - 可以监听消息并自动回复
   - 可以操作Bitable（需要配置）

2. **feishu-sdk-python** - 飞书Python SDK
   - 支持Bitable操作
   - 需要申请权限

## 推荐做法

请按以下步骤操作：

1. **在飞书中创建一个Bitable表格**
   - 打开飞书 → 新建 → 多维表格
   - 创建以下字段：
     - 编号（文本）
     - 标题（文本）
     - 标签（多选）
     - 状态（单选：已入库/已转录/已改写/已发布）
     - 来源（文本）
     - 入库时间（日期）
     - 视频路径（文本）
     - 转录路径（文本）
     - 内容摘要（文本）

2. **分享表格链接给我**
   - 点击右上角「分享」
   - 复制链接发送给我

3. **我配置自动同步**
   - 每当有新视频入库
   - 自动添加记录到Bitable
