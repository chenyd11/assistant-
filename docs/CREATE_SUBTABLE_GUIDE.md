## 🔧 创建子表步骤

### 第1步：找到创建入口
在Bitable左侧边栏，找到 **「+」** 按钮

```
┌─────────────────────────────┐
│  📋 短视频账号管理系统        │
│                              │
│  [数据表]                    │
│    ↳ 当前的主表             │
│                              │
│  [+] ← 点击这个按钮          │
│                              │
│  [仪表盘]                    │
│  [工作流]                    │
└─────────────────────────────┘
```

---

### 第2步：选择创建类型
点击「+」后，选择：**「新建数据表」**

```
弹出的菜单：
┌────────────────────────┐
│  新建数据表     ✓      │  ← 选这个
│  从Excel新建           │
│  导入Excel             │
│  从模板新建            │
│  ...                   │
└────────────────────────┘
```

---

### 第3步：命名子表
在顶部输入框修改名称：

**原名称：** 数据表2
**改为：** 改写版本库

```
┌──────────────────────────────────┐
│  [改写版本库]  [数据表]  [+]      │
│   ↑                           │
│   点击这里重命名               │
└──────────────────────────────────┘
```

---

### 第4步：添加字段
点击 **「+ 添加字段」** 按钮，依次添加以下字段：

| 序号 | 字段名称 | 字段类型 | 配置说明 |
|-----|---------|---------|---------|
| 1 | 版本编号 | 文本 | 默认 |
| 2 | 关联视频编号 | 文本 | 默认 |
| 3 | 平台 | 单选 | 选项：公众号、小红书、抖音 |
| 4 | 账号 | 文本 | 默认 |
| 5 | 版本标题 | 文本 | 默认 |
| 6 | 内容 | 长文本 | 默认 |
| 7 | 字数 | 数字 | 默认 |
| 8 | 标签 | 多选 | 选项：干货、避坑、职场、打工 |
| 9 | 创建时间 | 日期 | 默认 |
| 10 | 发布时间 | 日期 | 默认，可为空 |
| 11 | 发布链接 | 超链接 | 默认 |
| 12 | 状态 | 单选 | 选项：草稿、审核中、已发布、已删除 |
| 13 | 备注 | 文本 | 默认 |

---

## 💡 小技巧

### 如何切换子表？
点击左侧边栏的子表名称即可切换：
```
┌─────────────────┐
│ 数据表          │  ← 点击切换
│ 改写版本库      │  ← 点击切换  
└─────────────────┘
```

### 如何关联数据？
在「改写版本库」子表的「关联视频编号」字段，填写主表的视频编号（如：VID_2026_0001_76d70ac1），就能建立关联。
