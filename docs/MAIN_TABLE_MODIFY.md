# 主表（原始素材库）字段修改指南

## 📋 需要在飞书中手动修改的字段

### 1️⃣ 修改表格名称
**操作：**
1. 打开主表：https://fcnrncf2hy2l.feishu.cn/base/V87Cb06erar7kGsTFm0cR0JZnof
2. 点击左侧边栏的表格名称
3. 重命名为：**原始素材库**

---

### 2️⃣ 修改标签字段选项
**当前选项：** 教育、娱乐、科技、体育、美食、旅游、时尚、新闻

**改为：**
- 🟦 避坑干货
- 🟩 生活vlog  
- 🟧 工厂信息

**操作：**
1. 点击「标签」字段头
2. 选择「编辑字段」
3. 删除原有选项
4. 添加新选项（3个）

---

### 3️⃣ 字段重命名

| 原字段名 | 新字段名 |
|---------|---------|
| 来源 | 原链接 |
| 视频路径 | 视频 |
| 转录路径 | 转译结果 |
| 后续处理 | **删除** |

**操作：**
1. 双击字段头
2. 修改名称
3. 对于「后续处理」字段，点击字段头 → 删除字段

---

### 4️⃣ 修改「视频」字段类型
**当前：** 文本
**改为：** 附件（允许上传mp4文件）

**操作：**
1. 点击「视频」字段头（原"视频路径"）
2. 选择「修改字段类型」
3. 选择「附件」
4. 确认修改

---

```

视频: [附件]                          内容: （改写内容）
转译结果: （转录文字）
```

**如何查看关联：**
1. 在主表点击某条记录
2. 查看「关联视频编号」相同的子表记录
3. 或在子表中筛选「关联视频编号」字段
