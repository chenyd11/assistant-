# 抖音视频转多平台内容 - 使用指南

## 快速开始

### 方式一：全自动（推荐）

直接发送抖音视频文件给我，我会自动：
1. 提取音频并转录文字
2. 洗稿重写（保留核心信息，改变表达方式）
3. 生成3个平台版本
4. 推送到飞书文档
5. 返回文档链接

**使用步骤：**
```
1. 在抖音App中下载视频
2. 发送视频文件给我
3. 等待2-3分钟
4. 收到飞书文档链接
5. 在飞书文档中查看、编辑3个版本
6. 复制到字流，一键发布14平台
```

### 方式二：半自动（需要自定义）

如果你想控制中间过程：

**步骤1：视频转文字**
```bash
# 在OpenClaw工作目录执行
./scripts/process_douyin_video.sh /path/to/video.mp4
```

**步骤2：洗稿重写**
告诉我：
- 原始文本在哪里
- 你想要的风格（幽默/严肃/专业/口语化）
- 目标平台（可以只生成某几个版本）

**步骤3：推送到飞书**
```bash
python3 scripts/push_to_feishu.py "标题" "公众号内容" "小红书内容" "抖音内容"
```

## 问题排查

**Q: Whisper转录中文准确率低？**
A: 确保使用 `--language Chinese` 参数，或尝试更大的模型（medium/large）

**Q: 飞书文档创建失败？**
A: 检查飞书API权限，确保有文档创建权限

**Q: 内容风格不满意？**
A: 提供你喜欢的文章样例，我调整洗稿Prompt
