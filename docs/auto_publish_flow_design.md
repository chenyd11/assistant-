# 抖音视频转多平台内容 - 自动化流程设计

## 目标流程

```
用户输入 → 视频处理 → 内容生成 → 飞书文档 → 字流分发 → 14平台发布
```

## 详细步骤

### 阶段1：输入接收（已完成✅）
- 用户发送抖音视频文件
- 保存到本地 workspace/media/inbound/

### 阶段2：视频处理（已完成✅）
- ffmpeg 提取音频
- Whisper 语音转文字
- 输出原始文本

### 阶段3：内容生成（已完成✅）
- 洗稿重写（保留核心信息，改变表达方式）
- 生成3平台版本：
  - 公众号长文（1000-2000字）
  - 小红书图文（300-800字）
  - 抖音图文（100-300字）

### 阶段4：飞书文档推送（待配置）
- 创建飞书文档
- 写入3个版本内容
- 返回文档链接

### 阶段5：字流分发（待配置）
- 调用字流API创建草稿
- 或推送到字流支持的平台

### 阶段6：半自动发布（建议）
- 用户确认内容
- 一键发布到14平台

---

## 需要配置的内容

### 1. Agent配置（SOUL.md）
定义"洗稿助手"的角色：
- 职责：视频转文字、洗稿、多平台适配
- 风格：根据用户偏好定制
- 工具权限：飞书API、文件操作

### 2. 飞书API配置
- 创建飞书应用
- 获取 app_id / app_secret
- 配置文档创建权限

### 3. 字流API配置（如有）
- 获取API密钥
- 配置分发平台

### 4. 自动化触发
- 监听飞书消息（视频文件）
- 自动执行流程
