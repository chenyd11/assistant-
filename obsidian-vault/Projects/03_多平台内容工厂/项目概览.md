# 🏭 多平台内容工厂

## 🎯 项目目标

用 OpenClaw 跑通：给定内容 → 一键批量洗稿 → 生成适合多平台的稿件 → 自动发布。

## 🔄 工作流程

```
输入（视频/文章/选题）
    ↓
OpenClaw 处理
    ↓
├─→ 公众号版（深度长文）
├─→ 小红书版（emoji+配图）
└─→ 抖音图文版（短平快）
    ↓
自动发布 / 待审核发布
```

## 🛠️ 技术栈

- 视频处理：ffmpeg
- 语音转文字：Whisper
- 改写引擎：Kimi/GPT
- 发布：各平台API/自动化工具
- 调度：OpenClaw cron
